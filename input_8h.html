◆ PR3DKeyboardKey

enum PR3DKeyboardKey

KEYCODE MAPPINGS These are solely for the purpose of mapping to GLFW keycodes to abstract out GLFW so it does not need to be included in a project.

This also means the printable keys map to 7-bit ASCII, and the mappings follow a US keyboard layout

void pr3d_mouse_position ( double *  x_pos,

Gets the mouse cursor x,y positions and stores them in x_pos and y_pos.

Will return a screen position with 0,0 in the upper left if mouse capture mode is PR3D_MOUSE_CAPTURED, or an unbounded double if capture mode is PR3D_MOUSE_DISABLED.
